<threat_model>

## Trust Boundaries

| Boundary                 | Description                                                                                               |
| ------------------------ | --------------------------------------------------------------------------------------------------------- |
| stdin -> script          | Untrusted JSON input from Claude Code crosses here. Malformed, oversized, or malicious payloads possible. |
| script -> stdout         | Output consumed by Claude Code. Malformed XML could confuse the parser.                                   |
| script -> Perplexity API | Outbound network call with API key. Not in this plan (Plan 02).                                           |

## STRIDE Threat Register

| Threat ID | Category               | Component             | Disposition | Mitigation Plan                                                                                                                          |
| --------- | ---------------------- | --------------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| T-01-01   | Tampering              | stdin JSON input      | mitigate    | Zod .parse() rejects unexpected fields, validates types, enforces query min length                                                       |
| T-01-02   | Tampering              | XML output formatting | mitigate    | escapeXml() escapes &, <, >, " characters in titles and URLs                                                                             |
| T-01-03   | Information Disclosure | Logger output         | mitigate    | Logger never outputs API key values; only writes to stderr, never stdout                                                                 |
| T-01-04   | Tampering              | npm package install   | accept      | All packages are [ASSUMED] but are well-known ecosystem standards; slopcheck unavailable                                                 |
| T-01-SC   | Tampering              | npm/pip installs      | mitigate    | All packages verified as well-known ecosystem tools; slopcheck unavailable but all are [ASSUMED] with verified repo URLs per RESEARCH.md |

</threat_model>
