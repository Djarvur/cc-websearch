      - If tool_use, extract the tool name from event.input?.name and add to toolNames array
      - Return { toolNames }
    - On 'error' event (spawn failed): reject with error message including claude CLI not found hint
    - Set a reasonable timeout (not at spawn level — the vitest timeout handles this)
    - Note: the pluginDir is resolved relative to this test file: ../../.claude-plugin
