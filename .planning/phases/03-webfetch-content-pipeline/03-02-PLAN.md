## Trust Boundaries

| Boundary                  | Description                                     |
| ------------------------- | ----------------------------------------------- |
| Internet -> content.ts    | Untrusted HTML content from arbitrary web pages |
| content.ts -> Perplexity  | Extracted markdown sent to third-party API      |
| Perplexity -> webfetch.ts | Untrusted API response (answer text)            |

RESEARCH anti-pattern: never pass `runScripts: "dangerously"`                                                                                                                                 |

Low-value target (CLI tool, no persistent state).                                       |

SDK v0.29.0 verified.                                                                                                                                   |
| T-03-11   | Tampering         | Perplexity response content       | accept      | Response content is written directly to stdout (D-03). No further processing.
